# CMS React + Node.js

## current functionality.

- single page application + web api services
- full authentication & authorization
- full CRUD operations
- file upload

---

## How to run loacaly

> Client Side | Frontend App

```
  1. git clone project
  2. cd client
  3. npm install or yarn
  4. yarn start
```

> Server Side | Backend RESTful API

```
  1. git clone project
  2. cd server
  3. npm install
  4. npm run server
```
